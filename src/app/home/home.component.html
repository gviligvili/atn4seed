Angular 2 with NgRedux 2 & webpack Seed assembled by Tal Gvili

            The app uses Redux (NgRedux) as data flow.
            I tried to cover as much as many Redux aspects:
            -Actions
            -Reducers
            -Async
            -Entities (Schemas, Normalize and Denormalize)
            - Middlewares and Enhancers
            - Undo Redo History (See counter reducer).
            (Didn't include sagas,
            because seems to me that angular servicers can replace them)
            while trying to keep right practice.
            Each of the above covered with tests, to help you get going.

            The app also support HMR With angular2 and redux. (see Functionality below).

Tests

Test runner : Karma
Testing framework : Jasmine
Testing tools : Chai & sinon


For a single run with coverage : npm run test
*To see the coverage, after the test run - open coverage/html/index.html.


For a test watch : npm run watch:test
*Coverage was remove for testwatch. (because of the rapid runs).


The current seed contains various of tests, each in different niche (sync, async, reducers, actions,e2e) They can provide great examples for common needs.
